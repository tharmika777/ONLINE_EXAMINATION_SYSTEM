## 📌 Project Description
The Online Examination System is a console-based application developed using C++.  
It is designed to conduct an online MCQ examination where users can answer questions, and the system automatically evaluates responses and generates the final score.

This project helps in understanding core programming concepts like structures, arrays, loops, and conditional statements.

---

## 🚀 Features
- Multiple Choice Questions (MCQ) based exam
- Automatic answer checking
- Score calculation
- Displays final result instantly
- Simple and user-friendly console interface

## 🛠️ Technologies Used
- Programming Language: C++
- Concepts Used: Structure, Arrays, Loops, If-Else Statements
